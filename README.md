# 项目简介 python929

本文介绍一个基于Python的网络攻击行为可视化分析系统，该系统通过数据分析和前端可视化技术，实现网络攻击数据的直观展示。

## 项目概述

本系统使用了加拿大网络研究所提供的DDoS评价数据集（CIC-DDoS2019），对原始数据进行了深度处理，通过特征提取、时间片划分以及归一化等步骤，为用户提供了一个清晰的网络攻击行为可视化平台。

## 技术栈

### 前端
- Echarts
- jQuery

### 后端
- Flask
- Pandas

## 功能模块

系统主要包括以下功能模块：
- DDoS网络攻击数量统计
- 攻击流量时序统计
- 网络攻击导引图
- 网络攻击特性统计
- 部分PC遭受网络攻击占比统计

## 系统角色
- 数据处理与分析角色
- 可视化展示角色

## 运行环境

- Python3.8
- PyCharm

## 部署步骤
1. 环境配置：确保Python3.8及PyCharm环境搭建完成。
2. 数据准备：获取CIC-DDoS2019数据集并加载。
3. 系统运行：启动后端Flask服务和前端可视化界面。
